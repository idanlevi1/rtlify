# RTLify Playground

`BrokenDashboard.tsx` is a React component packed with intentional RTL violations — physical Tailwind classes, inline CSS, hardcoded Hebrew with unprotected numbers, and unflipped directional icons.

## Try it

```bash
cd examples/playground
npx rtlify check
```

Watch the linter catch every violation. Then try fixing them with the `/rtlify` slash command in Claude Code.
